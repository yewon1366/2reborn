핵심 기능

총무국이 겪는 모든 고통을
한 곳에서 해결합니다

수기 장부, 카톡 명단 공유, 감사 전날의 야근. 돈워리이 모두 대신합니다.

01
통합 행사 대시보드 → 바로가기
캘린더로 행사 일정을 한눈에 파악하고, 전년도 예산과 실시간 비교. 미입금·미검토 영수증 수를 즉시 확인.
02
3중 명단 대조기
신청자·입금내역·납부자 명단을 드래그앤드롭으로 올리면 4단계로 자동 분류. 모든 결과에 즉시 수정 버튼 제공.
03
AI OCR 영수증 판독
사진 한 장으로 품목·금액 자동 추출. 불확실한 글자는 하이라이트로 표시해 사람이 직접 확인 후 장부 반영.
04
권한 기반 협업 (RBAC)
국원은 업로드만, 국장은 수정·확정·최종승인. 데이터 무결성을 위한 결재라인으로 실수를 원천 차단.
05
감사 패키지 원클릭 추출
태그 하나로 행사별 파일 필터링. 영수증 사진부터 결산서까지 자동 폴더 구조로 ZIP 생성 후 즉시 다운로드.
06
클라우드 아카이브
#태그 시스템으로 모든 증빙 자료를 체계 관리. 다음 기수로의 인계도 링크 하나로 완료. 구글 드라이브처럼 쉽게.
